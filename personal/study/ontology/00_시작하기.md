# 온톨로지 스터디 시작하기

## 📖 학습 목표

- 온톨로지의 개념과 필요성 이해
- RDF/RDFS 데이터 모델 학습
- OWL (Web Ontology Language) 활용
- Protégé 도구로 온톨로지 설계
- 추론 엔진 활용 실습

## ✅ 학습 체크리스트

### Week 1: 온톨로지 기초
- [ ] 온톨로지 정의 및 역사
- [ ] 온톨로지 vs 택소노미 vs 데이터베이스
- [ ] 온톨로지의 실제 활용 사례 조사
- [ ] 기본 용어 정리 (클래스, 속성, 인스턴스, 관계)

### Week 2: RDF/RDFS
- [ ] RDF 트리플 구조 이해 (Subject-Predicate-Object)
- [ ] RDF 직렬화 형식 (Turtle, N-Triples, RDF/XML)
- [ ] RDFS 스키마 정의
- [ ] SPARQL 쿼리 기초

### Week 3: OWL
- [ ] OWL 2 개요 및 프로파일
- [ ] 클래스 정의 및 계층 구조
- [ ] 객체 속성 vs 데이터 속성
- [ ] 제약 조건 (Restrictions)
- [ ] 추론 규칙 (Axioms)

### Week 4: Protégé 실습
- [ ] Protégé 설치 및 인터페이스 익히기
- [ ] 간단한 도메인 온톨로지 설계 (예: 가족 관계)
- [ ] 추론기 실행 및 결과 확인
- [ ] 온톨로지 검증 및 디버깅

### Week 5-6: 실전 프로젝트
- [ ] 실제 도메인 선택 (예: 피트니스, 음악, 음식)
- [ ] 온톨로지 설계 및 구현
- [ ] Python에서 온톨로지 활용 (Owlready2)
- [ ] 추론 결과 활용 애플리케이션 개발

## 📝 실습 예제 주제

1. **가족 관계 온톨로지**
   - 클래스: Person, Male, Female
   - 속성: hasParent, hasChild, hasSibling
   - 추론: 조부모, 삼촌/이모 관계 자동 추론

2. **피트니스 도메인 온톨로지**
   - 클래스: Exercise, Equipment, Trainer, Member
   - 속성: requires, teaches, performs
   - 추론: 운동 프로그램 추천

3. **음식 온톨로지**
   - 클래스: Ingredient, Recipe, Dish, Cuisine
   - 속성: contains, origin, allergen
   - 추론: 알레르기 정보, 요리 추천
