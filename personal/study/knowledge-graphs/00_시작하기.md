## 🎯 핵심 Cypher 쿼리 예제

### 노드 생성
```cypher
CREATE (p:Person {name: 'Alice', age: 30})
CREATE (c:Company {name: 'TechCorp'})
CREATE (p)-[:WORKS_AT {since: 2020}]->(c)
```

### 패턴 매칭
```cypher
MATCH (p:Person)-[:WORKS_AT]->(c:Company)
WHERE c.name = 'TechCorp'
RETURN p.name, p.age
```

### 경로 찾기
```cypher
MATCH path = shortestPath(
  (a:Person {name: 'Alice'})-[*]-(b:Person {name: 'Bob'})
)
RETURN path
```

### 추천 시스템
```cypher
// Alice와 비슷한 관심사를 가진 사람들이 좋아하는 것
MATCH (alice:Person {name: 'Alice'})-[:LIKES]->(item:Item)
      <-[:LIKES]-(other:Person)-[:LIKES]->(recommendation:Item)
WHERE NOT (alice)-[:LIKES]->(recommendation)
RETURN recommendation.name, COUNT(*) AS score
ORDER BY score DESC
LIMIT 5
```
